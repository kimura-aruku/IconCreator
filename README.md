# ICO Converter

PNG、JPEG、SVG 画像から、複数サイズを含む ICO ファイルを生成する静的 Web アプリケーションです。

## 特徴

- 16、24、32、48、64、128、256 px のサイズを選択可能
- 選択した複数サイズを1つの ICO ファイルに格納
- インストールやビルドは不要
- 画像の読み込みから変換、ダウンロードまでブラウザ内で完結
- 外部サーバーへの画像アップロードなし

## 使い方

1. PNG、JPEG、または SVG ファイルをドロップするか、クリックして選択します。
2. ICO ファイルに含めるサイズを選択します。
3. `Generate .ico` を押します。
4. 生成された ICO ファイルがダウンロードされます。

## ローカルでの実行

`index.html` をWebブラウザで開いてください。Webサーバーや追加の依存関係は必要ありません。

## プライバシー

選択した画像はブラウザ内でのみ処理され、外部サーバーへ送信されません。本アプリケーションは外部フォントや外部JavaScriptを読み込みません。
